    a ← 0
    b ← 0
    c ← 0

    Tant Que n > 0 Faire :

        Si n >= 100 Alors
            n ← n – 100
            a ← a+ 1

        Sinon Si n >= 10 Alors
            n ← n – 10
            b ← b + 1

        Sinon
            n = n – 1
            c = c + 1

    Retourner [a, b, c]
